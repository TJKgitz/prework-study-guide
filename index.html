Prework Study Guide

HTML

  • The HEAD element contains information about the webpage.
  • title: required element defines the title for the webpage
  • style: element defines the CSS styles associated with a website
  • base: element defines the base URL for a website
  • link: element connects an external resource to the HTML document
  • meta: element defines metadata such as the character set, description, keywords, author
  • script: element embeds client-side scripts in the HTML document
  • The BODY element represents the visible content shown to the user.
  • h1, etc: headers should be numeric in order
  • img: element contains information about images that are displayed on website
  • src: location of the image displayed
  • alt: contains a text string that describes the appearance and function of the image
  • main: contains main document content
  • a: anchor element, works with href attribute to point to the URL link location
  • footer: element appears on the bottom of the page

CSS

  • Three ways to style a webpage using CSS
  • 1. Inline CSS: apply within an HTML tag using a style attribute
  • 2. Internal CSS: in the head apply to all or specified element
  • 3. External CSS: external file is linked using link element and placed in the head using rel and href attributes
  • within the css file, you can style out each selectors

Git

  • git branch: checks what branch we are on, maybe need to switch branches
  • git status: checks for changes in current branch, resolve any conflicts
  • git checkout main: to go to main branch before creating a feature branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stage all files
  • git commit -m "message": to commit changes with message to github
  • may need to do:
  • 1. git stash: remove changes on branch
  • 2. git merge origin main: copy from main branch
  • git pull origin main: verify we have all the latest changes
  • git push origin feature/branch-name: pushes to github
